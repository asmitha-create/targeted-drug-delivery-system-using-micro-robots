About the Project

The AI-Based Micro Robot Treatment Recommendation System is a futuristic healthcare platform designed to assist doctors and patients by analyzing symptoms and recommending the most suitable disease diagnosis, medicines, micro robots, and treatment methods.

This project integrates rule-based AI logic with a structured disease and micro robot database. It demonstrates how artificial intelligence can be applied in medical technology to improve accuracy, efficiency, and patient outcomes.

Key Features:

  • Rule-based AI symptom analysis
  • Comprehensive disease database
  • Micro robot recommendation system
  • Professional medical report generation
  • Responsive, futuristic hospital-style UI

This system is built using HTML5, CSS3, and JavaScript (ES6) only, without any backend or external APIs, making it lightweight and suitable for academic demonstrations.
